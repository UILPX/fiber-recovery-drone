# 硬件设计

## 地面机器人硬件

核心部件：

- 低速越障底盘：履带式优先，四轮差速可作为早期原型。
- 收卷盘：可拆卸，支持不同直径光纤盘。
- 导向机构：导向轮、排线器、线缆夹持器。
- 张力传感器：安装在导向轮或夹持路径上。
- 长度计量：编码器或计米轮。
- 控制器：负责电机、传感器和安全停机。
- 急停：本机实体急停 + 远程急停。

早期原型建议：

- 用现成电动底盘降低开发风险。
- 收卷电机选低速大扭矩，配合离合器或限矩结构。
- 张力控制优先于速度控制。

## 无人机硬件

核心部件：

- 稳定飞行平台。
- 云台相机或前下视相机。
- 轻量末端工具：导线钩、柔性夹爪、可释放线环。
- 螺旋桨保护和失联保护。

早期原型建议：

- 先用无人机做巡检，不急于挂载工具。
- 工具重量控制在飞行平台额定载荷的安全范围内。
- 末端工具做成可脱落或可释放，避免卡住后拖拽无人机。

## 接口草案

| 接口 | 数据 | 说明 |
| --- | --- | --- |
| 张力传感器 -> 控制器 | N 或相对值 | 用于恒张力控制和异常停机 |
| 编码器 -> 控制器 | 脉冲 | 估算回收长度和收卷速度 |
| 地面机器人 -> 控制台 | 位置、张力、长度、电量、状态 | 任务监控 |
| 无人机 -> 控制台 | 视频、位置、标记点、状态 | 巡检和解缠确认 |
| 控制台 -> 地面机器人 | 开始、暂停、速度、张力阈值 | 任务控制 |
| 控制台 -> 无人机 | 巡检点、观察点、返航、暂停 | 空中控制 |
