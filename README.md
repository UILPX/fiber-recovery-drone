# 光纤回收无人机项目

面向战后重建、灾后恢复和野外通信设施清理的无人化光纤回收系统。项目采用“地面机器人 + 无人机”协同方案：无人机负责识别和处理光纤缠绕、跨越、挂高等复杂情况，地面机器人负责牵引、导向、收卷、记录和运输。

## 项目目标

- 降低人员进入危险区域清理废弃光纤、通信线缆时的风险。
- 提高战后重建中道路、村镇、临时通信节点周边的线缆清理效率。
- 尽量完整回收可复用光纤，减少缠绕、拉断和二次污染。
- 建立可审计的数据记录：回收路线、长度估计、缠绕点、风险点、视频证据。

## 核心思路

系统由三部分组成：

1. 空中无人机
   - 巡检、建图、识别光纤走向。
   - 处理挂树、绕杆、跨沟、缠绕废墟等高处或复杂点位。
   - 用轻量夹爪、导线钩或可控释放机构辅助解缠。

2. 地面机器人
   - 搭载收卷盘、张力传感器、导向轮和线缆计米机构。
   - 沿线缓慢牵引和收卷，保持恒张力。
   - 搭载相机、激光雷达或深度相机识别地面障碍。

3. 操作与安全系统
   - 人在安全距离外监督。
   - 对疑似未爆物、危险建筑、通电线路、人员活动区域设置禁入逻辑。
   - 所有自动动作都保留人工暂停、退回和释放机制。

## 当前阶段

这是概念设计和工程启动阶段。建议先做一个低风险原型：

- 室内/封闭场地模拟光纤缠绕。
- 地面机器人完成恒张力收卷。
- 无人机只执行观察和简单拨线，不进入危险现场。
- 通过视频、张力和回收长度数据验证方案可行性。

## 非目标

- 不用于军事攻击、布雷、爆破、目标指示或对抗行动。
- 不让机器人主动接触疑似未爆弹药、带电高压线或结构不稳定建筑。
- 不以完全自主方式进入未经排查的战后高危区域。
