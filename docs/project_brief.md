# 项目构想

## 一句话描述

一套用于战后重建和灾后恢复的光纤回收机器人系统：无人机负责处理高处、缠绕和复杂路径，地面机器人负责稳定牵引、收卷和运输。

## 背景

战后重建现场可能存在废弃通信线缆、临时光纤、断裂线束和散落卷盘。这些线缆会影响道路清理、车辆通行、人员通行、通信重建和废料回收。人工清理面临高处作业、废墟作业、未爆物风险、绊倒风险和线缆突然回弹风险。

本项目把任务拆成两个角色：

- 无人机解决“人够不到、地面车过不去、线缆缠在高处”的问题。
- 地面机器人解决“长距离、重复、需要稳定张力和收卷”的问题。

## 典型场景

- 临时通信光纤横跨道路、沟渠或低矮建筑。
- 光纤缠绕在树枝、路牌、围栏、废墟钢筋或电线杆低处附件上。
- 大量废弃光纤散落在村镇道路、临时营地、机场跑道边缘或抢修通道。
- 需要回收可复用光纤卷盘，并记录损坏位置。

## 用户

- 战后重建工程队。
- 通信抢修队。
- 排雷和危险区域标记团队的辅助作业人员。
- 灾后基础设施恢复团队。

## 成功标准

- 单人可在安全距离外监督系统作业。
- 地面机器人能在可控张力下连续收卷至少 100 m 模拟光纤。
- 无人机能识别并辅助解除至少 3 类常见缠绕：树枝/杆件、围栏、废墟边缘。
- 系统能在检测到张力异常、视觉丢线、危险区域或通信中断时停机。
- 每次任务生成包含路线、视频片段、张力曲线和回收长度的记录。

## 关键假设

- 初期处理的是废弃或已确认断开的光纤，不处理带电线缆。
- 作业区域已完成基础安全排查，疑似未爆物区域由专业人员标记。
- 无人机不承担重载牵引，只做观察、拨线、解挂和导向。
- 主要牵引力由地面机器人提供，且通过张力闭环限制最大拉力。
