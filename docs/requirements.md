# 初版需求

## 功能需求

| ID | 需求 | 优先级 |
| --- | --- | --- |
| F-001 | 地面机器人能够夹持并收卷模拟光纤 | P0 |
| F-002 | 收卷系统能够读取张力并限制最大牵引力 | P0 |
| F-003 | 无人机能够巡检并标记疑似缠绕点 | P0 |
| F-004 | 操作台能够显示地面机器人、无人机和线缆状态 | P0 |
| F-005 | 系统能够在张力突增时自动暂停收卷 | P0 |
| F-006 | 无人机能够使用轻量工具拨开或抬起缠绕光纤 | P1 |
| F-007 | 系统能够估算已回收光纤长度 | P1 |
| F-008 | 系统能够导出任务报告 | P1 |
| F-009 | 支持人工接管无人机和地面机器人 | P1 |
| F-010 | 支持多段线缆任务队列 | P2 |

## 非功能需求

| ID | 需求 | 指标 |
| --- | --- | --- |
| NF-001 | 安全停机 | 急停后 1 s 内停止收卷电机 |
| NF-002 | 张力控制 | 可配置最大张力阈值 |
| NF-003 | 可维护性 | 收卷盘、导向轮和夹持器可现场快速更换 |
| NF-004 | 可审计性 | 保存视频、张力、长度、位置和操作日志 |
| NF-005 | 鲁棒性 | 通信中断时地面机器人停止收卷，无人机悬停或返航 |
| NF-006 | 便携性 | 两人可运输早期原型系统 |

## 约束

- 初期原型只在封闭场地测试。
- 初期处理轻质模拟光纤或废弃光纤，不处理电力线。
- 无人机末端工具不得有高速旋转、切割或破拆功能。
- 任何涉及战后危险区域的现场测试都必须由合规专业团队完成排查。

## 数据记录

每次任务至少记录：

- 任务 ID、时间、地点、操作员。
- 机器人轨迹和无人机轨迹。
- 已回收长度估计。
- 张力曲线。
- 缠绕点照片或视频片段。
- 停机原因和人工干预记录。
