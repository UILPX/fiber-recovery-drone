# 研发路线图

## 阶段 0：问题验证，1-2 周

- 访谈通信抢修、重建、排障或回收人员。
- 收集典型光纤缠绕照片和场景。
- 定义模拟测试场：树枝、杆件、围栏、废墟边缘、沟渠。
- 确定光纤样品、卷盘规格和最大安全张力。

产出：

- 场景库。
- 初版需求表。
- 测试场布置图。

## 阶段 1：地面收卷原型，2-4 周

- 搭建低速地面底盘或使用现成平台。
- 制作收卷盘、导向轮、张力传感器和编码器。
- 实现恒张力收卷控制。
- 完成 20 m、50 m、100 m 模拟光纤回收测试。

产出：

- 地面机器人原型。
- 张力控制代码。
- 测试报告。

## 阶段 2：无人机巡检与标记，2-4 周

- 使用商用无人机或开源飞控平台进行视觉巡检。
- 建立线缆和缠绕点标注数据。
- 实现操作台地图标记和视频记录。
- 先不做自动解缠，只做观察和人工确认。

产出：

- 巡检流程。
- 缠绕点数据集。
- 任务控制台雏形。

## 阶段 3：协同作业，3-6 周

- 地面机器人收卷时，无人机在前方观察。
- 张力异常自动触发暂停和缠绕点检查。
- 无人机尝试轻量拨线、抬线或导向动作。
- 完成封闭场地 3 类缠绕场景测试。

产出：

- 协同状态机。
- 解缠工具原型。
- 端到端演示。

## 阶段 4：现场级工程化，持续迭代

- 加固底盘和收卷机构。
- 增加防尘、防雨和易维护设计。
- 完善安全认证、飞行许可和现场作业规程。
- 与专业重建/排障团队联合评估。

产出：

- 工程样机。
- 安全手册。
- 试点部署计划。
