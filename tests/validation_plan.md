# 验证计划

## 原型测试目标

验证“地面机器人负责收卷，无人机负责识别和辅助解缠”的协同方案是否可行。

## 测试场景

| 场景 | 描述 | 通过标准 |
| --- | --- | --- |
| T-001 | 直线地面光纤 20 m | 完整收卷，张力无异常 |
| T-002 | 直线地面光纤 100 m | 长度误差可记录，收卷不卡滞 |
| T-003 | 光纤绕过低矮杆件 | 张力异常触发暂停 |
| T-004 | 光纤挂在树枝模拟物 | 无人机识别并辅助抬线 |
| T-005 | 光纤缠绕围栏 | 系统暂停并请求人工确认 |
| T-006 | 通信中断模拟 | 地面机器人停止收卷，无人机执行安全策略 |
| T-007 | 张力传感器异常 | 系统进入暂停或放弃状态 |
