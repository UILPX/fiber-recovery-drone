# 软件设计

## 软件模块

```text
software/
├── control/        # 地面机器人运动、收卷和张力控制
├── perception/     # 光纤识别、缠绕点检测、场景理解
├── mission/        # 协同状态机、任务编排和异常处理
├── console/        # 操作台 UI 和任务记录
└── simulation/     # 封闭场地和算法仿真
```

## 协同状态机草案

| 状态 | 说明 | 进入条件 | 退出条件 |
| --- | --- | --- | --- |
| IDLE | 待命 | 系统上电 | 开始任务 |
| SURVEY | 无人机巡检 | 任务开始 | 路径确认 |
| ACQUIRE_CABLE | 地面机器人夹持线缆 | 路径确认 | 夹持成功 |
| TEST_PULL | 低张力试拉 | 夹持成功 | 张力稳定 |
| RECOVER | 收卷回收 | 试拉通过 | 完成或异常 |
| UNTANGLE | 无人机辅助解缠 | 张力异常或视觉确认缠绕 | 解缠成功或放弃 |
| PAUSED | 暂停 | 人工暂停或风险触发 | 人工恢复 |
| ABORT | 任务放弃 | 危险、失联或人工终止 | 人工复位 |
| COMPLETE | 完成 | 线缆回收完成 | 归档 |

## 最小可行软件

- 地面机器人本地控制：
  - 手动前进/后退/停止。
  - 收卷电机开环速度控制。
  - 张力阈值停机。

- 操作台：
  - 显示张力、长度、速度、电量。
  - 显示无人机视频流或录制文件。
  - 标记缠绕点。
  - 导出任务日志。

- 数据格式：
  - JSONL 任务日志。
  - MP4 视频记录。
  - CSV 张力和长度曲线。

## 后续算法方向

- 光纤路径分割：结合颜色、细线检测和时序跟踪。
- 缠绕点识别：检测线缆急弯、交叉、遮挡和固定点。
- 张力异常分类：区分正常摩擦、缠绕、卡死和拉断。
- 地空协同规划：根据缠绕点位置决定无人机观察位和地面机器人暂停点。
